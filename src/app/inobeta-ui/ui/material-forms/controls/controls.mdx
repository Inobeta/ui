# Control recepies

## Autocomplete

```typescript
new IbMatAutocompleteControl({
  key: "country",
  label: "Country",
  options: [
    { value: "Italy" },
    { value: "Germany" },
    { value: "France" },
    { value: "Spain" },
  ],
});
```

## Button

```typescript
new IbMatButtonControl({
  key: "clear",
  label: "Clear",
  color: "accent",
  handler: (form) => form.reset(),
});
```

## Checkbox

```typescript
new IbMatCheckboxControl({
  key: "eggs",
  label: "Buy some eggs",
});
```

## Datepicker

```typescript
new IbMatDatepickerControl({
  type: "date",
  key: "departureAt",
  label: "Departure",
});
```

## Dropdown

```typescript
new IbMatDropdownControl({
  key: "theme",
  label: "Theme",
  value: "auto",
  options: [
    { key: "auto", value: "System default" },
    { key: "light", value: "Light" },
    { key: "dark", value: "Dark" },
  ],
});
```

## Radio

```typescript
new IbMatRadioControl({
  key: "newsletter",
  value: false,
  label: "Do you want to subscribe to the newsletter?",
  options: [
    { key: true, value: "Yes" },
    { key: false, value: "No" },
  ],
});
```
