## Override parameters

The parameters can be overriden by providing the following tokens

```typescript
@NgModule({
  imports: [
    /** ... */
    IbHttpModule,
  ],
  exports: [
    /** ... */
  ],
  declarations: [
    /** ... */
  ],
  providers: [
    { provide: "ibHttpGUILoginUrl", useValue: "..." },
    { provide: "ibHttpAPILoginUrl", useValue: "..." },
    { provide: "ibHttpAuthType", useValue: "..." },
    { provide: "ibHttpSessionStorageType", useValue: "..." },
    { provide: "ibHttpAPIRefreshUrl", useValue: "..." },
    { provide: "ibHttpJWTClaimsField", useValue: "..." },
    { provide: "ibHttpJWTRolesField", useValue: "..." },
  ],
})
export class AppModule {}
```
